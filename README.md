# Frontend Task

A frontend task to implement figma design

## Run Locally

Clone the project

```bash
  git clone https://github.com/isiakaabd/frontend-task.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
